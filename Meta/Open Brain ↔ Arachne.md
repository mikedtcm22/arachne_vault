# Open Brain ↔ Arachne

How the two memory systems divide labor and feed each other.

## The split
- **[[Open Brain]] — thought database.** Atomic, additive, recombinant. Individual observations, ideas, and tasks captured fast and embedded for semantic recall, so they can collide and recombine later. Granular memory.
- **This vault — system wiki.** Integrated, structured, durable. Systems and playbooks that have congealed from many thoughts into a coherent structure I maintain and interlink. Structural memory.

## The loop
1. Thoughts accumulate in Open Brain and cluster around a theme.
2. When a cluster stabilizes into a real structure, **promote** it into an Arachne system note here — and link back to the source thoughts.
3. Working a system note throws off new questions and observations — **capture** those back into Open Brain as fresh atomic thoughts.

## Rules of thumb
- A single thought goes to Open Brain.
- Something I maintain, revisit, and link earns a note here.
- A note here should reference the Open Brain cluster it grew from, so the lineage stays visible.
- The billable work itself, and ordinary life, belong in neither.
