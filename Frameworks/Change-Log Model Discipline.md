# Change-Log Model Discipline

Up: [[Arachne Intelligence]]

Modeling discipline: every model carries a **version-to-version reconciliation (change-log) tab**, so changes between versions are explicit and auditable. Removes silent drift and supports fast iteration — e.g., the projections model v2.0 in the [[ResFrac Candidacy]].
